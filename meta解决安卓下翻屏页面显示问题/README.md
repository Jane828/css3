发现meta标签功能太强大啦。

今天做了一个静态页面，在iphone 8 plus、iphone4下 显示正常，拿安卓手机测试下，发现高度不够，解决的办法是在html标签下添加如下代码，设置全屏预览就完美解决问题了。

```html
<!--设置全屏 -->
<meta name="full-screen" content="yes">
<meta name="x5-fullscreen" content="true">
```
